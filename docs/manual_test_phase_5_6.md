# Manual Test Checklist - Phase 5 to Phase 8

Usar un save nuevo o descartable cuando sea posible.

## Phase 5 (5A/5B/5C/5D)

- Construction:
  - iniciar mejora nivel 1,
  - validar timer,
  - acelerar con Focus,
  - cancelar y validar refund de materiales.
- Construction claim:
  - dejar completar y reclamar,
  - confirmar nivel del building y progreso diario relacionado.
- Caravan:
  - iniciar Caravan regional,
  - validar bloqueo de nuevas expediciones mientras está activa,
  - reclamar recompensas (gold, material regional, account XP).
- Outpost:
  - derrotar boss regional,
  - elegir bonus de outpost,
  - cambiar bonus y validar efecto actualizado.
- Class change:
  - usar early free respec antes de rebirth,
  - luego validar flujo de class change post-rebirth (cost/cooldown).

## Phase 6

- Traits:
  - encontrar/craftear Rare+ y validar trait visible.
- Context gear:
  - usar Equip Best para boss y validar preferencia por coverage relevante aunque baje Item Power bruto.
- Families:
  - equipar 2-3 items de una familia activa y validar resonance.
- Presets:
  - guardar Preset 1 y 2,
  - alterar gear,
  - re-equipar preset.
- Item lock:
  - bloquear item en inventario/equipped,
  - validar que sell/salvage quedan bloqueados hasta unlock.

## Phase 7

- Region diaries:
  - completar tareas de un diario regional,
  - reclamar reward,
  - validar efectos persistentes (material bonus/mastery bonus/título/trofeo según reward).
- Codex:
  - desbloquear trait/family discovery entries,
  - confirmar persistencia tras reload.
- Caravan mastery:
  - completar caravanas en una región,
  - reclamar tiers,
  - validar que bonuses afectan rewards/duración.

## Phase 8

- Regiones nuevas activas:
  - confirmar unlock/playability de `azure-vaults`, `stormglass-peaks`, `first-forge`.
- Materiales regionales Phase 8:
  - validar gain de `archiveGlyph`, `stormglassShard`, `oathEmber`.
- Collections/diaries extendidos:
  - validar summaries visibles y progreso en regiones 3-5.
- Soul Mark extensions:
  - comprar upgrades tardíos y validar efecto en economía/progresión.
- Account Rank extension:
  - verificar que rangos altos incrementan cap de Focus según tabla.

## Phase 9

- Event banner:
  - confirmar que aparece un evento activo con nombre, tiempo restante y barra de participación.
- Non-punitive participation:
  - completar expediciones (éxito y fracaso),
  - verificar que ambas avanzan participación del evento.
- Reward schedule:
  - completar objetivo de Tier 1,
  - reclamar reward desde banner,
  - verificar que no se puede reclamar dos veces el mismo tier.
- Temporary bonuses:
  - validar que durante el evento hay aumento temporal de mastery/regional materials según definición.
- Completion notifications opt-in:
  - habilitar toggle en Settings,
  - confirmar request de permiso del navegador,
  - con permiso otorgado, validar notificación para completion de Caravan/Construction,
  - con opt-in apagado, confirmar que no dispara notificaciones.
