# Relic Forge Idle - Development Roadmap

## Goal

Ship a strict MVP with hero-first progression and reincarnation loop.

## Week Plan (7 Days)

Day 1:

- Finalize docs and constants.
- Implement game types/content skeleton.
- Seeded RNG and state init.

Day 2:

- Expedition engine and rewards.
- XP/leveling and class passives.

Day 3:

- Loot system (rarity, affixes, compare).
- Inventory and equipment actions.

Day 4:

- Forge actions (craft/upgrade).
- Town buildings and economy.
- Vigor and offline caps.

Day 5:

- Contracts system and reset schedule.
- Reincarnation system and permanent upgrades.
- Save/import/export/reset safety.

Day 6:

- Mobile-first UI for all systems.
- First-session UX polish.

Day 7:

- Balance tuning.
- Test hardening.
- Release checklist pass.
- Awards UI polish only if all release gates are already green.

## Critical Path

1. Deterministic engine.
2. Loot + forge + town loop.
3. Reincarnation.
4. Contracts and vigor.
5. UI.

## Done When

- MVP is playable end-to-end.
- Release gates in `VERSION_1_0_DEFINITION.md` are met.
