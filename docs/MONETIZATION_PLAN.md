# Relic Forge Idle - Monetization Plan (Post-MVP Ready, Not Implemented)

## MVP Rule

No monetization implementation in v1.0.

No:

- premium currency
- payments
- ads
- rewarded ads
- purchase UI

## Future Monetization Direction

Allowed post-MVP directions:

- premium currency for optional convenience
- paid vigor refill
- rewarded ads for small vigor boosts
- supporter packs
- extra inventory tabs
- cosmetics

## Fairness Guardrails

- No pay-to-win power gates.
- Core progression must remain playable without spending.
- No daily streak punishment mechanics tied to purchases.

## Vigor Design Intent

MVP:

- Vigor is free and naturally regenerating.

Post-MVP:

- Optional paid refills can exist but cannot block non-paying progression.

## Inventory Monetization Note

MVP:

- Fixed inventory 30.

Post-MVP:

- Optional extra tabs are permitted, but not in v1.0.

## Done When

- Monetization is documented as future-only.
- MVP remains clean of payment systems.
